# Contributing

Thank you for your interest in this project.

This repository is provided primarily as a **portfolio and reference implementation**
of an RPA workflow using Python, PyAutoGUI and Pandas.

## Scope

Contributions are welcome as long as they remain aligned with the original purpose of the project:

- Documentation improvements
- Small refactors for readability or robustness
- Minor bug fixes
- Improvements related to safety and execution control

This project is **not intended to be a production-ready solution**.

## Guidelines

- Do not include real credentials, internal URLs or sensitive information
- Do not commit downloaded files, screenshots or execution artifacts
- Keep changes small and focused
- Avoid introducing unnecessary architectural complexity

## How to contribute

1. Fork the repository
2. Create a feature or fix branch
3. Open a pull request explaining what was changed and why

By contributing, you agree that your contributions will be licensed under the MIT License.
