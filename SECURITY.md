# Security Policy

## Supported Versions

This project is provided as a portfolio and reference implementation.
Security updates may be applied when relevant, but no formal support is guaranteed.

## Reporting a Vulnerability

If you identify a potential security issue, please avoid opening a public issue with sensitive details.

Instead, report it privately by contacting the repository owner.

## Usage and data safety

- Do not include real credentials, internal system URLs or sensitive data
- Do not commit downloaded files, screenshots or execution artifacts
- Use only authorized environments and accounts
- This project should not be used in production environments without proper review and adaptations
